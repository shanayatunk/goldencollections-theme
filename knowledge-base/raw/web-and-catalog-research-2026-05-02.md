## Content Opportunities Observed

- The store already has unusually strong long-tail category depth; SEO should emphasize taxonomy, compatibility, and regional terminology rather than only broad keywords.
- Product titles and tags already contain valuable search terms like Arangetram, Kuchipudi, Matil/Mattal/Mattel, Oddiyanam/Vaddanam/Kamarband, Namam/Thiruman/Tilak, Mukut/Kireedam, Hastham/Padam.
- Deity products often include sizes in variants, which can support idol-height compatibility guides.
- Deity jewellery needs a structured compatibility model: deity-specific, multi-deity, and common/general alankaram.
- Bharatanatyam set pages need a clean split between regular sets and premium real kemp sets.

## Content Inconsistencies Observed

- Some live pages still stated old trust, certificate, or stone-origin claims that owner later corrected.
- Some footer/search snippets still showed an old contact email; owner-confirmed email is `support@goldencollections.com`.
- Some product content uses gemstone-style wording; owner-confirmed public wording should be Kemp stones or Kempu stones.
- Some draft Varalakshmi products have placeholder-like price `Rs. 3.00` and no featured media. They are draft, so less urgent publicly, but they indicate catalog hygiene needs.
- Some active products have zero or negative inventory, especially older accessory items. This may affect merchandising and SEO if visible.

## Immediate Research Questions Generated

- How should real kemp be visually marked in menus, product titles, and collection pages?
- Which deity compatibility mapping should be built first?
- Which concise public story should replace the old heritage narrative?
- Which product types deserve separate buying guides vs glossary-only entries?
